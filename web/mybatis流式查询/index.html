Mybatis流式查询

November 11, 2024

1 背景#

当使用 MyBatis 查询大量数据时，用流式查询，可以显著降低内存占用，适用于大数据量导出。

2 核心参数与接口#

左转官网简介，不复制了

  • fetchSize
  • resultSetType
  • ResultHandler
  • ResultContext

3 使用示例#


@Select("SELECT id, name FROM user")
@Options(
        resultSetType = ResultSetType.FORWARD_ONLY,
        fetchSize = Integer.MIN_VALUE
)
void selectAllUsers(ResultHandler<User> handler);

sqlSession.select("selectAllUsers",new ResultHandler<User>() {
    @Override
    public void handleResult (ResultContext < ? extends User > context){
        User user = context.getResultObject();
        System.out.println("Processing: " + user.getName());
    }
});
